# 🏥 Testing Nurse — Тестирование медицинских сестёр

Мобильное приложение для прохождения квалификационных тестов медицинскими сёстрами. Поддерживает несколько тематических курсов, демо-режим, оплату доступа и админ-панель.

## Возможности

- **Регистрация и авторизация** — email/пароль через Supabase Auth, восстановление пароля
- **Выбор темы теста** — несколько тематических курсов с настраиваемым количеством вопросов
- **Демо-режим** — бесплатный пробный тест перед оплатой
- **Тестирование** — вопросы с 4 вариантами ответов, подсчёт результатов
- **Оплата доступа** — интеграция с ЮKassa для покупки курсов
- **Промокоды** — активация доступа по промокоду
- **Профиль** — смена пароля, удаление аккаунта
- **Админ-панель** — управление тестами, пользователями и настройками
- **Онбординг** — приветственные экраны для новых пользователей

## Архитектура

```
lib/
├── auth/                  # Авторизация (Supabase Auth)
├── backend/
│   ├── api_requests/      # API вызовы (ЮKassa, Edge Functions)
│   └── supabase/          # Таблицы БД: users, questionnaire, subject, orders
├── custom_code/
│   └── actions/           # Кастомные действия
├── admin_panel/           # Админ-панель
├── home_page/             # Главная — выбор темы теста
├── course_selection/      # Выбор курса и оплата
├── question/              # Экран вопроса с вариантами ответов
├── results/               # Результаты тестирования
├── profile/               # Профиль пользователя
├── registration/          # Регистрация, вход, восстановление пароля
└── flutter_flow/          # Утилиты, тема, навигация
```

## Стек технологий

- **Flutter** — кроссплатформенный UI
- **FlutterFlow** — визуальное построение UI и навигации
- **Supabase** — база данных (PostgreSQL), авторизация, Edge Functions
- **ЮKassa** — приём платежей за доступ к курсам

## База данных

| Таблица | Описание |
|---------|----------|
| `users` | Пользователи: email, статус оплаты, срок доступа |
| `subject` | Темы тестов с количеством вопросов |
| `questionnaire` | Вопросы: текст, 4 варианта, правильный ответ |
| `orders` | Заказы на оплату курсов |
| `global` | Глобальные настройки приложения |

## Custom Actions

| Файл | Описание |
|------|----------|
| `supabase_get_user.dart` | Получение данных текущего пользователя |
| `update_supabase_password.dart` | Смена пароля через Supabase Auth |

## Серверная часть (Supabase)

- **Edge Functions** — удаление аккаунта пользователя (`delete-user`)
- **Auth** — регистрация, вход, смена пароля

## Запуск

```bash
flutter pub get
flutter run
```

Требуется настроить:
- Supabase URL и Anon Key в `lib/backend/supabase/supabase.dart`
- Ключ ЮKassa в `lib/backend/api_requests/api_calls.dart`
